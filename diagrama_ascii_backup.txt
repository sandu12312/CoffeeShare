FLUXURI DE DATE:
================

CLIENT → SECURITY:
• Mobile App ──→ Firebase Auth (Secure Authentication)
• Mobile App ──→ SSL Pinning (SSL Handshake)
• Mobile App ──→ Secure Storage (Local Data Cache)
• Admin Panel ──→ Firebase Auth (Admin Authentication)

SECURITY → BACKEND:
• Firebase Auth ══→ Firestore (User Data Sync)
• SSL Pinning ══→ Cloud Functions (Secure API Calls)
• Secure Storage ══→ Firestore (Cache Synchronization)

BACKEND INTERNAL:
• Firestore ←→ Cloud Functions (Real-time Updates)
• Cloud Functions ──→ Storage (Asset Management)
• Cloud Functions ──→ Analytics (Event Tracking)

EXTERNAL INTEGRATIONS:
• Mobile App ──→ Google Maps (Location Services)
• Mobile App ──→ Stripe (Payment Processing)
• Mobile App ──→ Sentry (Error Reporting)
• Cloud Functions ══→ Stripe (Payment Validation)

CORE FEATURES:
• Mobile App ══→ QR Management (QR Code Operations)
• Mobile App ══→ Subscription Engine (Subscription Management)
• Mobile App ══→ Analytics (Statistics Retrieval)
• QR Management ══→ Cloud Functions (Token Validation)
• Subscription Engine ══→ Firestore (Data Persistence)
